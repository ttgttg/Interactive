WWW vs Internet

  • The Internet allows computers to communicate with each other, while the WWW allows users to access information.
  • The Internet supports communication, entertainment, education — the WWW is mainly for browsing.
  • WWW is a set of protocols by Tim Berners-Lee:
    • HTTP → how data is transferred
    • URL → how resources are addressed
    • HTML → how content is structured
  • He was trying to solve the problem of scientists not being able to easily share information across systems.

Affordances of "Gesture"

What actions are possible (or feel natural) when using your body — hands, movement, posture — to interact with a system.
